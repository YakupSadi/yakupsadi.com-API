## Create new project
- ``` npm init -y ```
- ``` npm install express ```
- ``` npm install --save-dev nodemon ```
- ``` npm start ```
- ``` npm install mongoose --save ```
- ``` npm install dotenv --save ```
- ``` npm install jsonwebtoken ```
-  ``` npm install bcryptjs ```
-  ``` npm install cookie-parser ```

## .env
- ``` PORT         = 3000 ```
- ``` JWT_SECRET   = String ```
- ``` JWT_LIFETIME = 30d ```
